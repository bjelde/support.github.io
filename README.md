# tidy.

tidy. is an iOS app that helps users clean up screenshot clutter quickly.

Instead of scrolling through the entire Photos library, tidy. shows screenshots in a simple swipe flow. Users can save screenshots they want to keep, send unwanted screenshots to the dustpan, and empty the dustpan when they are ready.

## Features

- Review screenshots one by one
- Swipe to save or send screenshots to the dustpan
- Review saved screenshots
- Review dustpan screenshots before deleting
- Batch delete screenshots into Recently Deleted
- Track cleanup progress
- Optional weekly reminder notifications

## How It Works

- `Swipe right` to save a screenshot
- `Swipe left` to send a screenshot to the dustpan
- Open the saved section to review screenshots you kept
- Open the dustpan to review screenshots before deleting
- Empty the dustpan to move screenshots to Recently Deleted in Photos

## Privacy

tidy. uses photo library access only to help users review and manage screenshots they choose.

The app does not require an account.
